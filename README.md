# Calebsons-Web3-Escrow-Marketplace-Infrastructure-Suite
A full‑stack Web3 escrow marketplace built with Solidity, Foundry, Next.js, Wagmi, RainbowKit, Ethers.js, and a TypeScript GraphQL backend. Features on‑chain escrow, dispute resolution, indexing, and a modern dApp UI with complete infra and CI/CD.
